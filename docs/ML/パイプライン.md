# ML パイプライン設計

## 概要

- レビューデータから映画ベクトルを作成 -> 近似最近傍法で、5 件ずつ抽出する

## パイプライン

1. 入稿処理(レビューデータ取得)
2. 正規化処理
3. ベクトル化処理
4. 近似最近傍法で類似映画データ作成

## メモ

- TMDB との比較をベースラインとする
- 検証ツール: airflow, kedro, mlflow, kubeflow, bigquery, ai platform
- データのバージョン管理をどうするか？
- パイプラインは、kedro を使う
- 汎用的なパイプラインツールとしては、kedro は不足しているので、今後 airflow を検討
- 実験管理で、mlflow の導入も検討
