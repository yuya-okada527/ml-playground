# FE 機能設計

## コンシューマ FE

コンシューマ向けの映画機能を提供する。

### 機能一覧

- 映画情報機能
  - 検索
  - 詳細
- レコメンド機能
  - 類似映画紹介
- ユーザ機能
  - ユーザ情報保持
- お気に入り機能
  - 登録
  - 削除

### 優先度

- 優先 B
  - 映画情報機能
  - レコメンド機能
- 優先 C
  - ユーザ機能
  - お気に入り機能

## 入稿ツール

運営用の機能を提供する。

### 機能一覧

- 外部映画情報検索機能
- 映画情報登録機能
- 認証機能

### 優先度

- 優先 A
  - 外部映画情報検索機能
- 優先 B
  - 映画情報登録機能
- 優先 D
  - 認証機能
